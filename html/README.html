AI Processing Pipeline

Three-pass evaluation: each pass isolated at temperature=0 to minimise variance.

1

Vision Transcription

Vision model reads handwriting directly into plain text. No OCR middleware. Confidence score assigned for legibility.

// Image → Groq/Claude vision → Plain text transcript (temp=0)
2

Semantic Analysis

Text model evaluates conceptual coverage, keyword relevance, coherence, completeness, and accuracy against the model answer.

// Transcript + model answer → 5-dimensional semantic scores (temp=0)
3

Mark Award

Text model translates semantic scores into numeric marks using rubric weights and education-level criteria.

// Semantic scores + rubric + level → final marks + feedback (temp=0)

Semantic Scoring Framework

Beginner

Compassion floor 5%
Focus on basic understanding

Intermediate

Balanced evaluation
No compassion floor

Expert

Strict accuracy requirements
No floor

Dimension Weight Measures
Conceptual Coverage25–35%Core concepts understood
Grammar Coherence10–20%Structure and flow
Keyword Relevance15–25%Domain terminology usage
Completeness20%All required parts addressed
Accuracy Score10–20%Factual correctness

TypeScript Interface

interface SemanticScore {
  conceptualCoverage: number   // Core concepts understanding
  grammarCoherence:   number   // Structure and flow
  keywordRelevance:   number   // Domain terminology
  completeness:       number   // All required parts
  accuracyScore:      number   // Factual correctness
  composite:          number   // Weighted total score
}

Usage Guide

1

Create an Exam

  1. Navigate to Create Exam
  2. Upload question paper (PDF or image)
  3. Click Extract Questions with AI to auto-populate
  4. Upload answer key and extract model answers
  5. Review and adjust questions / rubrics, then save
2

Evaluate Student Answers

  1. Go to Evaluate page and select exam
  2. Enter student details (name, roll number)
  3. Upload answer sheet (PDF or image)
  4. Choose AI provider — Groq for speed, Claude for complexity
  5. Click Start Evaluation and monitor live progress
3

Review & Approve

  1. Navigate to Review page
  2. Inspect AI evaluations per question
  3. Adjust marks if needed and add reviewer comments
  4. Approve or request revision

Advanced Features

Confidence-Based Evaluation

AI confidence scores flag low-legibility answers for human review. Evaluation adapts based on confidence levels.

Multi-Page Document Support

Automatic page detection, cross-page answer merging, and efficient handling of 100+ page documents.

Error Resilience

Retry logic with exponential backoff, graceful handling of unreadable content, and comprehensive error reporting.

Performance Optimization

Parallel processing, efficient image compression, background task processing, and per-second progress updates.
